## Spacing 

| Name | Size | Uses |
| :--- | :--- | :--- |
| **Spacing XS** | 8px | **Small Elements:** Space between small icons and text. Padding for small containers (icon buttons, table cells) and spacing between repeated elements (button groups). |
| **Spacing S** | 16px | **Internal Padding:** Padding within input components and cards. Vertical spacing between related elements like a title and its description. |
| **Spacing M** | 24px | **Component Elevation:** Space between triggers and elevated elements (e.g., dropdown buttons and menus). |
| **Spacing L** | 32px | **Large Containers:** Padding for larger component containers (buttons, tracking steppers). Spacing between large icons and content. |
| **Spacing XL** | 48px | **Structural Spacing:** Vertical spacing between major elements in complex cards or larger UI components. |
| **Spacing XXL** | 64px | **Layout Layout:** Space between the top of the page and the header, or major sections of page content. |
| **Spacing XXXL** | 80px | **Page Alignment:** Alignment and breathing room within larger pieces of high-level content. |
| **Spacing Giant** | 96px | **Hero Sections:** Maximum alignment space for main containers and hero headers. |
